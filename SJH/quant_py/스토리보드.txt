총괄 Agent >>
Agent.md 파일에 해당 프로젝트에 전체적인 구조나 이런것들을 기재 하는 곳이고, 퀀트투자 포트폴리오를 웹 프로그램으로 만들거야
디자이너 Agent >
style.md 에이전트 하나 만들어줘 여기에는 디자인 관련한 내용이 기재될꺼야. 전체 톤은 벍은 느낌이 드는 상큼한 디자인으로 해줘
개발자 Agent (dev.md) >
개발에 관한 일은 dev.md 파일에 기재할꺼야. 구조 수정이나 기술적인 부분을 여기에 기재해줘. 이 파일을 바탕으로 구조 변경이나 수정이 있으면 참고할 파일이야.
